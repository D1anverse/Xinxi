# 🎓 大学生恋爱交友平台 - 开发计划

> **项目名称**: SoulMatch (暂定)  
> **核心理念**: 价值观匹配，而非外表筛选  
> **开发时间**: 2026-03-12 启动  
> **目标用户**: 在校大学生

---

## 💡 核心设计理念

### 与传统交友软件的区别

| 传统软件 | SoulMatch |
|---------|-----------|
| 外表优先（滑动照片） | 价值观优先 |
| 即时匹配 | 深度匹配算法 |
| 短期关系导向 | 长期关系导向 |
| 用户主动筛选 | 算法智能推荐 |

### 价值观匹配维度

1. **人生目标** - 职业规划、生活愿景
2. **性格特质** - 内向/外向、理性/感性
3. **兴趣爱好** - 共同话题基础
4. **恋爱观念** - 关系期待、相处模式
5. **生活方式** - 作息、消费、社交习惯

## 🧮 匹配算法设计

### 基础公式

```
匹配度 = Σ(维度权重 × 维度相似度)

维度相似度 = 1 - (|用户 A 得分 - 用户 B 得分 | / 最大分差)
```

### 权重设计

| 维度 | 基础权重 | 用户可调整 |
|------|---------|-----------|
| 人生目标 | 30% | ✅ |
| 性格特质 | 25% | ✅ |
| 恋爱观念 | 25% | ✅ |
| 兴趣爱好 | 10% | ✅ |
| 生活方式 | 10% | ✅ |

### 匹配流程

1. 用户完成问卷 → 生成价值观向量
2. 筛选条件（年龄/学校/性别等）
3. 计算与候选用户的匹配度
4. 按匹配度排序
5. 每日推荐 Top 5
