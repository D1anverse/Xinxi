# SoulMatch Frontend - 前端开发计划

## 🎯 技术选型

### 方案：Vue 3 + Vite + 微信小程序

**选择理由**:
- Vue 3 学习曲线平缓，开发效率高
- Vite 构建速度快，热更新体验好
- 微信小程序覆盖大学生用户群体
- 可复用代码逻辑 (Taro/uni-app)

---

## 📁 项目结构

```
frontend/
├── web/                    # Web 端 (Vue 3)
│   ├── src/
│   │   ├── components/     # 通用组件
│   │   ├── views/          # 页面视图
│   │   ├── router/         # 路由配置
│   │   ├── stores/         # Pinia 状态管理
│   │   ├── api/            # API 调用
│   │   ├── utils/          # 工具函数
│   │   └── styles/         # 样式文件
│   ├── public/
│   └── package.json
│
├── miniprogram/            # 微信小程序
│   ├── pages/
│   ├── components/
│   ├── utils/
│   └── app.json
│
└── shared/                 # 共享代码
    ├── types/              # TypeScript 类型定义
    ├── constants/          # 常量配置
    └── validators/         # 表单验证
```

## 🎨 设计规范

### 颜色

| 用途 | 颜色 | Hex |
|------|------|-----|
| 主色 | 樱花粉 | `#FF6B8A` |
| 辅色 | 薰衣草紫 | `#9B7EBD` |
| 背景 | 米白 | `#F8F5F2` |
| 文字 | 深灰 | `#333333` |
| 成功 | 绿色 | `#52C41A` |
| 警告 | 橙色 | `#FA8C16` |
| 错误 | 红色 | `#F5222D` |

### 字体

- 标题：18-24px
- 正文：14-16px
- 辅助文字：12px
